## 2026-04-16

### 技术选型
- **决定**：Next.js 14（App Router）+ TypeScript + Tailwind CSS + Vercel 部署
- **原因**：可长期维护，SEO 友好，后续接 Claude API 方便，与 Cursor/Claude Code 工作流兼容

### 页面结构
- **决定**：首页单页 7 sections + 独立项目详情页（/work/[slug]）
- **原因**：访客既可快速扫描首页，又可深入查看项目细节；独立页面便于 SEO 和链接分享

### AI 对话
- **决定**：当前跳过 API 接入，预留接口，Hero 区和 AI Exploration 用模拟静态回答
- **原因**：先把视觉和内容做稳，API 接入是后续独立任务

### 内容管理
- **决定**：项目详情和文章用 MDX，首页静态内容用 TypeScript data 文件
- **原因**：MDX 支持富文本 + 自定义组件，data 文件修改简单直接

### 项目重要性排序
- **决定**：Kaleido IDE 是第一优先代表作，先完成 Kaleido 详情页，其余两个复用模板
- **原因**：Kaleido 信息量最丰富，最能体现系统设计能力
